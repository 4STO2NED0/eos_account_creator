FAQ

Why do I need this service?

The way EOS works is that new accounts can only be created by someone with an existing account. If you have a friend who has an EOS account, you can ask him to create one for you. If you don't, this website can be your friend for a small fee.

Currently, only cryptocurrency payment is accepted. We accept Bitcoin, Bitcoin Cash, Ethereum and Litecoin. Fiat payment via credit card will be integrated if there is enough demand. Please understand that we can currently only offer this service to customers outside of the EU (european union).

Why is EOS not accepted?

If you already have an EOS account, you don't need this tool. You can use the excellent EOS Toolkit by GenerEOS. If you currently have EOS on an exchange only, it's not possible to use it for payment. You have to withdraw it to your own wallet first. The reason is that other cryptocurrencies let you create an individual receiving address per payment. On EOS, this is not possible, one has to use the memo field in the transfer. As far as I know, most exchanges don't let you use a custom memo when withdrawing.
